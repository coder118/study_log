
    Arrays.stream(arr).forEach(System.out::println);


함수();  -> 함수 실행
함수 -> 함수 자체

forEach내부에 있는 함수를 이용해서 출력하라는 뜻.

stream에 넘길 매개변수의 개수는 '일반적으로' 하나이다.

Stream은 일회성이다. 한번 사용하면 닫혀서 다시 사용할 수없다.
사용하고 싶은 함수가 있다. 거기서 단 한번만 사용하면 되는건데 굳이 외부에 함수를 선언할 필요가 있을까?
=> 일회성 함수를 사용해서 Stream 내부에 넣어줄 수있다. 

형태는 ()->{} 형태를 가지고 있다.
