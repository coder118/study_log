
# 문제

Stirng 식이 입력된다. x끼리 연산하고 상수끼리 연산해라.

# 처음 든 생각

String을 쪼개야 한다. 
1.  \+ 로 나누어서 배열에 저장한다.
2. 배열을 순회하며 문자열 x가 들어있는 값을 찾는다.
3. x문자열 앞의 자리를 뽑아서 정수형으로 바꾼다. x문자열이 한자리일 경우 1을 반환

# 문제 발생

테스트 케이스를 진행하는데 자꾸 문제가 생겼다. 80%는 통과되는데 자꾸 20%가 막혔다. 

# 해결

문제를 다시 읽어보니 x 앞에 1이 있을때 생략해준다는 조건을 깜빡했다. 
해당 조건문을 추가해줌으로써 해결했다.

    import java.util.Arrays;
    class Solution {
        public String solution(String polynomial) {

            String answer ="";

            Object[] arr =Arrays.stream(polynomial.split(" \\+ ")).toArray();

            int xcount = 0;

            int num_count = 0;

            for(Object i : arr){
                String a = String.valueOf(i);
                if(a.contains("x")){
                    if(a.equals("x")){
                        xcount +=1;
                        continue;
                    }
                    String num = a.substring(0,a.indexOf("x"));
                    xcount += Integer.parseInt(num);
                }else{
                    num_count+=Integer.parseInt(a);
                }
            }

            if (xcount != 0 && num_count == 0) {
                if(xcount ==1 ){
                    answer = "x";
                }else{
                    answer=String.format("%dx", xcount);
                }

            } else if (xcount == 0 && num_count !=0) {
                answer=String.format("%d", num_count);
            } else {
                if(xcount ==1 ){
                    answer=String.format("x + %d",  num_count);
                }else{
                    answer=String.format("%dx + %d", xcount, num_count);
                }

            }

            return answer.toString();
        }
    }

# 개선점 및 배운점
코드가 너무 쓸데 없이 길다. Arrays.stream을 사용할 필요도 없었다. 굳이 Object 객체를 받아서 String으로 바꿔주는 과정을 지워도 된다. 

String.format을 사용하지 않고 그냥 문자열끼리 + 연산을 하면된다.
자바를 더 빨리 손에 익게 해야겠다.


    String[] arr = polynomial.split(" \\+ ");
