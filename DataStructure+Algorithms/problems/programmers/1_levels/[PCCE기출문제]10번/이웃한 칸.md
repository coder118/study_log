# 문제 정의

- 동서남북에 위치한 색깔을 비교해야 한다.

1. 범위를 벗어나는 경우를 생각해봐야 한다.
2. 중심좌표에서 동서남북 위치 좌표를 구하는 방법을 생각해야 한다.
3. 어떻게 색깔을 비교해야 할지 생각해봐야 한다.

질문

1. 어떻게 범위를 벗어나지 않게 할 수 있을까?, 무엇을 사용해야 할까?
2. 정사각형 기준으로 동서남북을 구하는 식이 있을까?
3. 색깔은 배열 내에 문자열로 이루어져있다 어떻게 비교할 수 있을까?

답

1. 조건문을 사용해서 좌표를 벗어나려하면 차단한다.
2. 북쪽은 h-1, 남쪽은 h+1, 서쪽은 w-1,동쪽은 w+1이다.
3. equals를 사용해보자.


# 코드 구현

    String base_color = board[h][w];

    int width = board.length;
    int height = board.length;

    int count  = 0;

    if(h-1>=0){
        //북쪽이다.
        //0보다 작지 않아야 한다.=> 0 이상이야 한다.
        if(board[h-1][w].equals(base_color)){
            count++;
        }
    }
    if(h+1<height){
        //남쪽
        //h+1이 총 높이보다 높아서는 안된다.=> height가 더 커야 한다.
            if(board[h+1][w].equals(base_color)){
            count++;
        }
    }
    if(w-1>=0){
        //서쪽
        //0보다 작으면 안된다.=> 0이상이어야 한다.
            if(board[h][w-1].equals(base_color)){
            count++;
        }
    }
    if(w+1<width){
        //동쪽
        //총 width보다 커서는 안된다. => width보다 작아야 한다.
            if(board[h][w+1].equals(base_color)){
            count++;
        }
    }


반복문을 사용하면 더 간단하게 풀 수 있었을 것 같다. 이 문제는 주어진 의사코드가 있었는데 그대로 코드를 작성해도 문제를 풀 수 있다. 
<br>
살펴본 결과 내가 한 풀이를 반복문으로 간결하게 표현한 것이다.

    int n = board.length;
    
    int count =0;
    
    int[] dh = {0,1,-1,0};
    int[] dw = {1,0,0,-1};
    
    for(int i =0; i<4; i++){
        int h_check = h+dh[i];
        int w_check = w+dw[i];
        
        if(h_check>=0 && h_check<n 
            w_check>=0 && w_check<n){
            if(board[h][w].equals(board[h_check][w_check])){
                count++;
            }
        }
        
    }        

# 배운점

일일히 조건을 다 적는 방식이 아직까지는 사용되지만 조건이 조금만 더 많아져도 할 수 없는 방법이다. 그렇기에, 반복문을 활용해서 비교하는 방법에 익숙해져야 할 것 같다.
