
# 문제 정의

- 전화번호부를 나타내는 문자열 배열 phone_book을 입력받아 전화번호가 다른 전화번호의 접두어인지 확인하고 접두어인 경우 false를 반환하고, 접두어가 아닌 경우 true를 반환하는 문제이다.

- phone_book에 존재하는 전화번호를 key로 하고, 해당 전화번호의 인덱스를 value로 하는 Map을 선언.
- 반복문을 통해 phone_book 배열을 순회하여 각 전화번호와 인덱스를 map에 추가한다.
- 두 번째 반복문에서는 다시 phone_book 배열을 순회하여 각 전화번호의 접두어가 다른 전화번호의 전체 부분인지 확인한다.


# 코드 
    Map<String, Integer> map = new HashMap<>();

    // 모든 phone 번호 hashMap에 담기
    for (int i = 0; i < phone_book.length; i++) {
        map.put(phone_book[i], i);
    }

    for (int i = 0; i < phone_book.length; i++) {
        // 전화번호의 길이만큼 순회
        for (int j = 0; j < phone_book[i].length(); j++) {
            // 해당 전화번호의 접두어가 key로 가진 경우
            if (map.containsKey(phone_book[i].substring(0,j))) {
                return false;
            }
        }
    }

    return true;

----

1. 두 번째 반복문의 내부 반복문에서 현재 전화번호(phone_book[i])의 길이만큼 반복한다.
2. phone_book[i].substring(0, j) : 현재 전화번호의 첫 번째 글자부터 j 번째 글자까지 부분 문자열로 나눈다.
3. containsKey() 메서드를 통해 map에 현재 전화번호의 부분문자열을 키로 포함하고 있는지 체크한다.
4. 포함하고 있다면 false를 반환하고, 반복문을 무사히 끝마친 경우 true를 반환한다.
