
# 문제 정의

1. 세로(아래)로 이동 -> a[0][0]=1 -> a[1][0]=2 -> a[2][0]=3 같이 첫번째 인덱스 +1 증가
2. 가로(오른쪽)로 이동 ->  [4][1]=6 -> [4][2]=9 -> [4][3]=8 -> [4][4]=9 와 같이 두번째 인덱스 +1 증가
3. 대각선(왼쪽위)으로 이동 -> a[4][4] -> a[3][3]=10 -> a[2][2]=11 -> a[1][1]=10 와 같이 두 인덱스가 -1씩 이동 

# 코드

    class Solution {
        public int[] solution(int n) {
            int max = n*(n+1)/2; //등비수열합
            int[][] arr = new int[n][n];
            int[] answer = new int[max];

            // 시작 지점
            int x = -1, y = 0;
            int num = 1;

            for (int i = 0; i < n; ++i) {
                for (int j = i; j < n; ++j) {
                    if (i % 3 == 0) {
                        ++x; // 세로
                    } else if (i % 3 == 1) {
                        ++y; // 가로
                    } else if (i % 3 == 2) {
                        // 대각선
                        --x;
                        --y;
                    }
                    arr[x][y] = num++;
                }
            }

            int idx=0;
            for(int i=0; i<arr.length; i++){
                for(int j=0; j<=i;j++){
                    answer[idx] = arr[i][j];
                    idx++;
                }
            }

            return answer;
        }
    }


각 동작마다 행해지는 숫자가 1개씩 줄어든다. (j)동작은 3개이므로 3을 나눠서 나머지로 구분해 동작을 실행해준다. 마지막으로 2차원 배열을 1차원배열에 넣어서 반환한다.
