

# 문제 정의

사용자들의 수가 0시부터 24시까지 주어질때 필요한 서버의 개수를 구하는 문제이다. 문제에서 주어지는 조건은 사람이 m명 이상일 때부터 서버가 1개씩 늘어나고 해당 서버는 k시간 후에 없어진다는 조건

# 코드

    class Solution {
        static int[] server;
        //사람 명수 m, 서버 지속 시간 k
        public int solution(int[] players, int m, int k) {
            int answer = 0;
            server = new int[24];
            for(int i = 0; i < 24; i++){
                //현재 필요한 서버 수 계산
                int needServer = 0;
                if(players[i] >= m){
                    if(players[i] % m == 0){
                        needServer += ((players[i] - m) / m) + 1;
                    }
                    else needServer += ((players[i] - m) / m) + 1;
                }
                
                //현재 가용가능한 서버 개수 구하기
                int serverCount = 0;
                for(int j = Math.max(0, i - k + 1); j < i; j++){
                    serverCount += server[j];
                }
                
                //필요한 서버 수 계산
                int add = Math.max(0, needServer - serverCount);
                server[i] = add;
                answer += add;

            }
            return answer;
        }
    }

1. 현재 인원수를 기준으로 필요한 서버수를 계산, m 명을 기준으로 서버가 몇개나 필요한지 계산을 해주어야 한다.
2. m명일때는 따로 서버 증설이 필요하지 않기 때문에 players[i] - m 을 기준으로 계산
3. 현재 가용가능한 서버의 개수 카운트해야함
4. 현재 필요한 서버 - 현재 가용가능한 서버 를 구하고 해당 값을 servers[i] 에 넣어주고 그만큼 증설했다고 정답인 answer에 더해주었다.
