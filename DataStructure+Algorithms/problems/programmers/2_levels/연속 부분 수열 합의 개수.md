
# 문제 정의

기존의 주어진 배열을 2배로 늘려 탐색을 진행했다. HashSet을 사용해서 중복값을 배제


# 코드

    int answer = 0;

    HashSet<Integer> set = new HashSet<Integer>();
    ArrayList<Integer> lst = new ArrayList<Integer>();
    for (int i = 0; i<2; i++) {
        for (int element : elements) {
            lst.add(element);
        }
    }
    for (int i = 0; i < elements.length; i++) {
        for (int j = 1; j <= elements.length; j++) {
            List<Integer> subLst = lst.subList(i, i+j);
            int sum = 0;
            for (int num : subLst)
                sum += num;
            set.add(sum);
        }
    }
    answer = set.size();

> 코드 흐름

1. 주어진 배열을 리스트로 변환해 2배로 만듬
2. 기존 배열 길이 만큼 이중 탐색을 진행하며, 부분합을 진행한다.
3. 부분합의 결과는 HashSet에 추가한다.
4. 결과적으로 HashSet의 크기를 정답 처리한다.
