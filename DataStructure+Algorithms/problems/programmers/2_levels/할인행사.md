    public boolean isMatch(HashMap<String, Integer> map) {
        for(String key : map.keySet()) {
            if(map.get(key) != 0)
                return false;
        }
        return true;
    }


# 구체적 구현 방법

1. 원하는 제품의 이름과 수량 정리
    - HashMap을 사용해, 원하는 제품 이름과 수량을 중복되지 않게 저장.
2. 초기 10일 구간의 확인
    - discount 배열의 처음 10일간의 제품 목록을 HashMap과 비교
    - discount에 포함된 제품이 원하는 제품일 경우, 해당 제품의 수량을 하나 줄인다.
3. 날짜 이동
    - discount의 배열에서 1씩 이동하며 확인.
    - 새로운 날짜의 제품을 할인 제품 목록에서 수량을 하나 줄인다.
    - 제외되는 날짜의 제품의 수량을 하나 늘린다.
4. 필요 제품 충족 여부 확인
    - HashMap의 모든 값이 0인 경우 원하는 제품을 모두 구매한 것.
