# 문제 정의

- 점프하는 방법 : k칸을 앞으로 점프 => k 만큼 건전지 소비
- 순간이동 :(현재까지 온 거리) x 2 => 건전지 소비 안해도 됨

> **n만큼 떨어진 거리를 점프로 이동하는 거리를 최소로 하려고 한다.**

순간이동으로의 **이동을 최대로 사용**하고 점프로 이동하는 것을 최소로 사용해야 한다. 
<br>
<br>
n은 10억 이하 => n^2 복잡도를 가지지 않는다.

===

# 코드 

    int ans =0; //사용한 건전지 수
    while(n!=0){ 
        //n을 2로 나눈 나머지가 1일 경우 건전지를 사용해서 이동
        if(n%2==1){
            ans++;//건전지 사용
        }    
        n/=2;//순간이동 하기 전 위치로
    }
    return ans;

======
