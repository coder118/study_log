# 문제 정의

캐시 사이즈 크기로 배열을 만든다.
캐시 내부에 지금 searching 하는 값이 있으면 1초 없으면 5초이다.
캐시는 큐의 형태로 동작.

1. 캐시 큐 내부에값이 없으면 총 time에 5를 더한다.
2. 캐시 큐 내부값이 있다면 time에 1을 더한다.
2-1. 항상 첫 값은 빼고 새로 값을 넣는다.

---

    ArrayDeque<String> cacheQ = new ArrayDeque<>();
    
    int len = cities.length;
    int TotalTime= 0;
    for(int i = 0; i<len; i++){
        if(!cacheQ.isEmpty() && cacheQ.size()==cacheSize){// 캐시내부에 값이 있어야 하고 캐시 사이즈를 넘어서 안된다.
            String searchValue = cities[i].toLowerCase();
            if(cacheQ.remove(searchValue)){
                cacheQ.offer(searchValue);
                TotalTime +=1;
            }else{
                cacheQ.poll();
                cacheQ.offer(searchValue); 
                TotalTime +=5;
            }
            
        }else{//캐시가 비었다면
            cacheQ.offer(cities[i].toLowerCase());
            TotalTime +=5;
        }

    }

조건문에 캐시는 비어있지 않고 항상 사이즈는 주어진 cacheSize와 동일해야 하는 조건이 필요했다.


# 해결책

조건을 너무 꼬아서 생각했었다. 단순하게 캐시가 비어있지 않으며 사이즈가 동일해야 바뀌는 것이 아니라

1. 캐시에 있는지 없는지 확인한다.
2. 캐시에 없을 경우에는 
    1. 캐시가 꽉차있는 경우 맨 앞자리를 지우고 채운다.
    2. 캐시가 안 차있는 경우 그냥 add로 값을 채운다. 


    ArrayDeque<String> cacheQ = new ArrayDeque<>();
        
    int len = cities.length;
    int t= 0;
    if(cacheSize == 0){ //캐시 크기가 0이면
        return cities.length * 5;
    }
    for(int i =0; i< len ; i++){
        String city = cities[i].toLowerCase();
        if(cacheQ.contains(city)){
            cacheQ.remove(city);
            cacheQ.offer(city);
            t+=1;
        }else{
            if(cacheQ.size()==cacheSize){
                cacheQ.poll();
            }
            
            cacheQ.offer(city);
            t+=5;
        
        }

    }
