# 문제 정의

1. n의 이진수를 구하고 1의 개수를 샌다.
2. n 보다 큰 값의 이진수를 구해서 1의 개수를 카운팅한다.
3. 1,2의 개수가 동일한 값이 정답

이진수를 카운트하는 자바 함수를 몰라서 직접 구현했다.

# 코드 구현

    class Solution {
        public int solution(int n) { 
            int target = search(n);
            int count =0;
            int num = n+1;
            while(true){
                if(target==count){
                    break;
                }
                count=search(num);
                System.out.println(count);
                num++;
            }
            
            
            return num;
        }
        int search(int n){
            int count=0;
            while(n==0){
                if(n%2==1){
                    count++;
                }
                n=n/2;
            }
            return count;
        }
    }

로직 자체에는 문제가 없지만 더 간결하게 만들고 싶어서 자바에서 제공하는 함수를 사용해서 구현했다. 

    // n의 이진수 표현에서의 1의 개수를 세는 변수
    int count = Integer.bitCount(n);
    // n보다 큰 다음 정수부터 시작
    int answer = n + 1;

    while (true) {
        // 1의 개수가 n과 같아지면 반복문 종료
        if (count == Integer.bitCount(answer)) {
            break;
        }
        
        // answer 증가
        answer++;
    }
   
동일한 로직이지만 bitCount를 사용하자마자 코드가 간결해졌다. 

## 배운점

가독성적인 측면을 위해서라도 자바에서 제공하는 라이브러리를 사용하는 것이 좋다는 생각이 들었다.
