# 문제 정의


- 1씩 더하며 일일이 찾는 대신, 2진수의 규칙 패턴을 사용해야 한다. 
- 비트 차이가 2개 이하가 되는 가장 작은 f(x)는 가장 하위 비트(오른쪽)부터 보았을 때 처음 나오는 0을 1로 바꾸고, 그 바로 오른쪽의 1을 0으로 바꾸는 규칙을 가진다.

# 코드



    import java.util.*;

    class Solution {
        public long[] solution(long[] numbers) {
            long[] answer = new long[numbers.length];
            
            for (int i = 0; i<numbers.length; i++) {
                long result = numbers[i];
                String target = Long.toBinaryString(numbers[i]);
                
                if (result % 2 == 0) {
                    answer[i] = result + 1;
                } else {
                    int idx = target.lastIndexOf("0");
                    
                    if (idx == -1) {
                        String tmp = "10" + target.substring(1, target.length());
                        answer[i] = Long.parseLong(tmp, 2);
                    } else {
                        String tmp = target.substring(0, idx) + "10" + target.substring(idx + 2, target.length());
                        answer[i] = Long.parseLong(tmp, 2);
                    }
                }
            }
            return answer;
        }
    }

1 . Long.toBinaryString(num)과 Long.toString(num, 2)을 통해 쉽게 비트 변환.
2. lastIndexOf을 활용하여 String에서 특정 값의 마지막 인덱스를 반환. 만족하는 값이 없다면 -1을 반환.
3. substring을 활용하여 String에 대한 슬라이싱이 가능.
4. Long.parseLong(bit, 2)을 통해 비트를 다시 정수로 변환이 가능.
