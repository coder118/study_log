# 사고 과정 정리

매번 복잡하게 생각하는 것 같아서 간단히 정리했다.

문제 한줄 정리 - 대소 비교 

> 해결할 부분

1. 대소 비교인데 문자열로 주어짐
2. 리턴 값을 최소, 최대 형태로 되돌려야 한다.

1번 문제는 문자열을 정수로 바꿔주면 되고 2번 문제는 정수로 바꿨을때 최소 최대를 찾고 문자열을 반환하면 된다.


# 코드 풀이

<details>
<summary>코드 확인 버튼</summary>


    int[] compare_arr = Arrays.stream(s.split(" ")).mapToInt(Integer::parseInt).toArray();
    
    int max = Arrays.stream(compare_arr).max().orElseThrow();
    int min = Arrays.stream(compare_arr).min().orElseThrow();
    
    answer = min + " "+ max;
    
    return answer;

</details>

위에서 말한 풀이 과정 그대로 스트림을 사용해서 구현했다.

## 다른 사람 풀이

<details>
<summary>코드 확인 버튼</summary>

    String[] tmp = str.split(" ");
    int min, max, n;
    min = max = Integer.parseInt(tmp[0]);
    for (int i = 1; i < tmp.length; i++) {
            n = Integer.parseInt(tmp[i]);
        if(min > n) min = n;
        if(max < n) max = n;
    }

    return min + " " + max;


최소 최대를 첫번째 값으로 설정하고 반복을 돌며 확인하는 방법을 사용


    String[] array = str.split(" ");
    int[] arrayInt = new int[array.length];

    for (int i = 0; i < arrayInt.length; i++) {
        arrayInt[i] = Integer.parseInt(array[i]);
    }

    Arrays.sort(arrayInt);
    String result = arrayInt[0] + " " + arrayInt[arrayInt.length - 1];

arrayInt는 str의 범위를 넘을 수 없으니 초기화해준다. 그리고 오름차순 정렬을 하면 제일 왼쪽은 최소 오른쪽은 최대값이 나올 것이기 때문에 인덱스로 선택한다.

</details>
