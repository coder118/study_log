

# 문제 정의

1. 그래프 초기화
2. 양방향 간선 추가
3. 입력된 간선을 하나씩 제거하면서 그래프를 두 개의 그룹으로 분리
4. dfs 함수

# 코드


    static ArrayList<Integer>[] graph;
    static int min;

    public int solution(int n, int[][] wires) {
    graph = new ArrayList[n + 1];
    min = Integer.MAX_VALUE;

    // 그래프 ArrayList 초기화. 노드 개수만큼 ArrayList 생성
    for (int i = 1; i <= n; i++) {
        graph[i] = new ArrayList<>();
    }

    // 양방향 간선 구조이므로 두 번 add를 해준다
    for (int i = 0; i < wires.length; i++) {
        int v1 = wires[i][0];
        int v2 = wires[i][1];
        graph[v1].add(v2);
        graph[v2].add(v1);
    }

    for (int i = 0; i < wires.length; i++) {
        int v1 = wires[i][0];
        int v2 = wires[i][1];

        boolean[] visited = new boolean[n + 1];

        // 해당 간선을 그래프에서 제거
        graph[v1].remove(Integer.valueOf(v2));
        graph[v2].remove(Integer.valueOf(v1));

        int cnt = dfs(1, visited); // 임의의 시작점에서 dfs 탐색

        int diff = Math.abs(cnt - (n - cnt));
        min = Math.min(min, diff);

        // 그래프에 다시 간선 추가
        graph[v1].add(v2);
        graph[v2].add(v1);
    }

        return min;
    }

    static int dfs(int v, boolean[] visited) {
        visited[v] = true;
        int cnt = 1;

        for (int next : graph[v]) {
            if (!visited[next]) {
                cnt += dfs(next, visited);
            }
        }

        return cnt;
    }


> 구체적 설명

1. 그래프 초기화
graph는 노드 개수 n에 따른 그래프의 인접 리스트를 저장하기 위한 배열이다. min은 트리의 최소 크기이다

2. 양방향 간선 추가
wirees 배열을 순회하면서 간선 정보를 graph 배열에 추가하고, add를 두 번 해준다.

3. 입력된 간선을 하나씩 제거하면서 그래프를 두 개의 그룹으로 분리

visited 배열은 방문 여부를 나타내기 위한 배열이며, 각 노드의 방문 상태를 저장한다.
전력망을 두 부분으로 나눌 때, 주어진 간선 중 하나를 끊어 두 부분으로 분리한다.
dfs 함수를 재귀적으로 호출하여 시작점에서부터 각 그룹의 노드 개수를 계산하고 그룹의 노드 개수 차이를 구하고, 최소 트리 크기인 min을 갱신한다. 이후, 제거한 단선을 다시 graph에 추가하여 다음 간선을 제거하고 계산을 진행하는 방식이다.

4. dfs 함수
현재 노드 v를 방문처리하고, 노드 개수를 세는 변수인 cnt를 1로 초기화한다. graph[v]에 저장된 인접한 노드들을 순회하면서 방문하지 않은 노드들을 탐색한다. 마무리로, 방문하지 않은 인접 노드 next를 재귀적으로 방문하며 cnt 업데이트해주고 탐색이 끝나면 cnt 값 반환한다.
