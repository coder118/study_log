# 문제 정의 

- skill 은 1 ~ 26 길이가 될 수 있다는 것에 유의 해야 한다.
- skill_trees 에서 skill 이 아닌 문자열을 모두 제거했고 제거된 문자열을 skill 의 문자열 배열 순서와 비교하여 일치하면 count 를 증가시켰다.

# 코드


    int answer = 0;

    for (int i = 0; i < skill_trees.length; i++) {
        skill_trees[i] = skill_trees[i].replaceAll("[^"+skill+"]","");
        boolean flag = false;
        for (int j = 0; j < skill_trees[i].length(); j++) {
            if(skill.charAt(j) != skill_trees[i].charAt(j)){
                flag = true;
                break;
            }
        }

        if(flag == false){
            answer++;
        }
    }
    return answer;

- replaceAll("[^"+skill+"]",""); 이 정규식을 이용하면 훨씬 편하게 문제를 해결할 수 있다.
