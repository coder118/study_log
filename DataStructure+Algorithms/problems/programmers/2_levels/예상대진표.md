
# 문제 정의

- 이 문제는 n명의 참가자 중 a와 b가 서로 만나게 되는 라운드를 반환하는 문제다.
- a와 b는 참가 번호를 뜻하며, n을 사용하지 않고도 간단히 문제를 해결할 수 있다.

# 코드 흐름

- a와 b가 같아질 때까지 a와 b를 각각 2로 나눈 몫과 나머지를 더하는 연산을 반복
- 이는 각 참가자 별로 둘로 짝지어 게임을 치르기 때문에 라운드가 종료할 때마다 절반의 참가자가 다음 라운드로 진출하는 구조
- 따라서, a와 b는 만날 때까지 이긴다고 가정했기 때문에 절반씩 나누어 몫과 나머지를 더하면 다음 라운드에 진출한 것으로 취급할 수 있다.


# 코드 

    public int solution(int n, int a, int b) {
        int answer = 0;
        
        // a와 b가 다르면 반복
        while (a != b) {
            // a와 b를 각각 2로 나눈 몫과 나머지와 더함
            a = a / 2 + a % 2;
            b = b / 2 + b % 2;

            // answer 증가(라운드 증가)
            answer++;
        }

        return answer;
    }
