# 문제 정의

- 남은 작업량이 큰 작업부터 처리하는 것이 피로도를 줄이는 데 효과적이다. 
- 남은 작업량을 우선순위 큐에 저장하고, 가장 큰 작업량을 가진 작업을 우선적으로 처리하는 방식이 필요하다.


# 코드


    import java.util.*;
    class Solution {
        public long solution(int n, int[] works) {
            long answer = 0;
            PriorityQueue<Integer> queue = new PriorityQueue<>(Comparator.reverseOrder());
            
            for(int i = 0; i < works.length; i++) {
                queue.offer(works[i]);
            }
            
            while(n > 0) {
                int tmp = queue.poll();
                if(tmp == 0) break;
                tmp--;
                n--;
                queue.offer(tmp);
            }
            
            while(!queue.isEmpty()) {
                int t = queue.poll();
                answer += t * t;
            }
            return answer;
        }
    }

1. PriorityQeueue에 현재 남은 작업량을 넣어준다(Comparator 통해 내림차순 정렬)

2. 내림차순 정렬을 했기 때문에 가장 큰 값이 나오게 된다 작업량을 1개감소 시키고 큐에 다시 넣어준다.

3. 위의 과정을 n의 값이 0이 되도록 반복한다

4. 마지막으로 큐에 남은 값들을 제곱해서 더해주면 완료
