# 문제정의

1. visited는 노드 방문 여부를 체크하는 boolean형 배열을 만든다.
2. for 반복문을 통해 모든 노드를 탐색
3. 방문하지 않은 노드일 경우에는 dfs 메소드로 넘어감
4. visited를 true로 설정해준 후 네트워크 연결 정보가 담긴 computers 배열을 탐색
5. 방문한 적 없는 노드이면서 computers 배열 값이 1이라 연결되어 있다면 dfs 메소드로 넘어간다
6. 해당 로직을 통해 네트워크 개수를 answer++을 통해 파악할 수 있다.


# 코드

    class Solution {
        
        static boolean[] visited;
        
        public int solution(int n, int[][] computers) {
            int answer = 0;
            
            visited = new boolean[n];
            
            for(int i=0; i<n; i++){
                if(!visited[i]){
                    answer++;
                    dfs(i, computers);
                }
            }
            return answer;
        }
        
        public void dfs(int x, int[][] arr){
            visited[x] = true;
            for(int i=0; i<arr.length; i++){
                if(!visited[i] && arr[x][i] == 1){
                    dfs(i, arr);
                }
            }
        }
    }
