# 문제 정의

아래 칸으로 내려갈 때, 바로 왼쪽 위 또는 오른쪽 위 숫자 중 큰 것을 골라 현재 숫자에 더한다.
꼭대기부터 바닥까지 이 과정을 반복한 뒤, 맨 마지막 줄에서 가장 큰 값을 찾는 문제이다.


# 코드


    class Solution {
        public int solution(int[][] triangle) {
            // 아래층부터 꼭대기 층(0)의 바로 위층까지 반복
            for (int i = triangle.length - 2; i >= 0; i--) {
                for (int j = 0; j < triangle[i].length; j++) {
                    // 바로 아래와 오른쪽 아래 칸 중 큰 값을 현재 칸에 더함
                    int max = Math.max(triangle[i+1][j], triangle[i+1][j+1]);
                    triangle[i][j] += max;
                }
            }
            
            // 꼭대기에 저장된 최댓값 반환
            return triangle[0][0];
        }
    }

1. 아래에서 위로 올라가며 합을 구한다.
2. 삼각형 맨 아래층부터 시작합니다.
3. 윗줄의 숫자는 바로 아래층의 왼쪽과 오른쪽 숫자 중 더 큰 쪽과 합친다.
4. 위 합을 원래 숫자에 덮어쓴다 
5. 이 과정을 꼭대기까지 반복한다.
6. 마지막에 꼭대기(triangle[0][0])에 남은 숫자가 가장 큰 합.
