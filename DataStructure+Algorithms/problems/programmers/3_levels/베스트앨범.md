# 문제 정의

> 저장되는 데이터 구조
key(genre) : list[0,1,2]이런식으로 저장이된다. 하지만 인덱스가 고유 번호면 정렬시에 고유번호를 잃게 된다. 
<br>

=>**리스트를 이차원 배열로 세팅**
<br> 
<br> 
classic : [[0,200],[1,292]...]

=====
# 코드


    HashMap<String,List> genreMap = new HashMap<>();
    HashMap<String,Integer> playSum = new HashMap<>();
    
    for(int i=0; i<genres.length; i++){
        if(genreMap.containsKey(genres[i])){
            genreMap.get(genres[i]).add(new int[]{i, plays[i]}); //기존의 장르가 있을때 값을 추가한다.
            Integer sum = playSum.get(genres[i])+plays[i];
            
            playSum.put(genres[i],sum);
            continue;
        }
        
        genreMap.put(genres[i],new ArrayList());//처음 들어오는 장르를 넣는다.
        playSum.put(genres[i],plays[i]);
        
        genreMap.get(genres[i]).add(new int[]{i, plays[i]});
    }

classic : [[0,200],[1,292]...] 이런 구조를 가지는 코드를 작성함 만약, 이미 저장된 장르라면 리스트 내부의 그 다음 리스트를 넣어준다.


    //playSum의 values에서 어떤 key가 가장 최대 값을 가지고 있는지 출력한다.
    List<String> sortedGenres = new ArrayList<>(playSum.keySet());
    sortedGenres.sort((a,b)-> playSum.get(b)-playSum.get(a));

playSum의 합이 큰 값을 순서대로 해서 장르를 정렬해서 리스트에 넣어둔다. 리스트에는 조회수가 많은 장르 순서대로 저장된다

    List<Integer> answerList = new ArrayList<>();
    for(String genre : sortedGenres){
        List<int[]> list = genreMap.get(genre);
        list.sort((a, b) -> b[1] == a[1] ? a[0] - b[0] : b[1] - a[1]);
        for (int i = 0; i < Math.min(list.size(), 2);i++) {
            answerList.add(list.get(i)[0]);
        }
    }

    return answerList;

정렬된 장르 순서(sortedGenres)로 genreMap에 접근해서 고유번호를 출력한다. 출력할때 2개 값을 제한해서 answerList에 값을 넣는다. 

# comparator 정렬

    List<String> sortedGenres = new ArrayList<>(playSum.keySet());
    sortedGenres.sort((a,b)-> playSum.get(b)-playSum.get(a));

리스트에 담긴 것은 Key(장르 이름)이지만, 정렬 기준(Comparator)은 Map의 데이터(Value)를 참조한다. 
예를 들어보자면, ["classic", "pop", "jazz"] 같은 장르 이름(String)만 저장된다. 
<br> 
<br> 
sort 메서드가 실행될때 리스트안의 두 요소를 가지고 와서 비교한다. pop과 jazz를 비교할때 playSum에 접근해서 숫자를 비교해서 정렬을 시켜준다. 
<br> 
<br> 
- 오름차순 (작은 순): (a, b) -> a - b
- 내림차순 (큰 순): (a, b) -> b - a

# sortedGenres를 사용해서 정렬

    List<int[]> list = genreMap.get(genre);
    list.sort((a, b) -> b[1] == a[1] ? a[0] - b[0] : b[1] - a[1]);
    for (int i = 0; i < Math.min(list.size(), 2);i++) {
        answerList.add(list.get(i)[0]);
    }

- a[0], b[0] : 곡 번호 (고유 번호)

- a[1], b[1] : 재생 횟수

인덱스 1로 접근해서 재생횟수가 같은지 확인하고 같다면 그대로 유지하고 같지 않다면 큰 순서대로 정렬하게 만들었다. 

> Math.min의 사용이유

장르당 최대 2곡이라는 말이 있는데, 단순히 2를 사용해도 될 것 같지만, 곡이 1개뿐인 장르에서 list.get(1)을 호출하려다 에러(IndexOutOfBounds)가 나게 된다. 그걸 방지하기 위해 구현하였다.
