[문제로](https://www.acmicpc.net/problem/2798)

# 문제 정의

총 5개의 숫자가 주어졌을때 아래와 같은 인덱스 범위를 계산하면 3자리 조합의 모든 합을 구하고 그 중 M보다 작은 숫자를 찾을 수 있다.  

    012/013/014/023/024/034/123/124/134/234

1. 첫 번째 숫자를 변경하지 않고 나머지 2개의 숫자를 변경한다.
2. 총 3개의 숫자의 합이 M을 넘어가면 그 전 숫자를 출력한다.

# 코드 문제 풀이

    public static void main(String[] args) throws IOException {
            BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
            StringTokenizer st = new StringTokenizer(br.readLine());

            int N = Integer.parseInt(st.nextToken());
            int M = Integer.parseInt(st.nextToken());

            int[] arr = new int[N];
            st = new StringTokenizer(br.readLine());
            for (int i = 0; i < N; i++) {
                arr[i] = Integer.parseInt(st.nextToken());
            }
            Arrays.sort(arr);

            
            int prev = 0;
            int sum =0;
            boolean check = false;
            int ans = 0;
            for (int f = 0; f <= N/2; f++) {
                for (int s = f + 1; s <= f + 3; s++) {
                    for (int t = s + 1; t < N; t++) {
                        prev = ans;
                        sum =0;
                    sum+= arr[f]+arr[s]+arr[t];
                        if (ans > M) {
                            check =true;
                            break;
                        } else if (sum == M) {
                            check = true;
                            System.out.print(sum);
                            break;
                        }
                        if (sum < M) {
                            ans = Math.max(prev, sum);
                        }

                    }
                    if(check) break;
                }
                if(check) break;
            }

            System.out.print(prev);
        }

# 피드백

값을 처음부터 차근차근 더하다가 M보다 클 경우 그 전 값이 가장 M에 가까운 최대값이라 생각했다.근데 그것보다 3자리 조합의 모든 숫자 값의 합을 구하고 최종적으로 M을 넘지 않는 max값을 구해야 한다는 생각이 들었다. 
<br>
<br>

첫 번째 반복문의 범위를 N/2를 사용했었는데 3개를 고르기 때문에 N-2까지만 순회하고, 두번째는 그보다 하나 작은 N-1, 세번쨰는 N번째까지로 수정을 해주었다. 

## 수정 코드

    for (int f = 0; f <N-2; f++) {
        for (int s = f + 1; s <= N-1; s++) {
            for (int t = s + 1; t < N; t++) {

                sum= arr[f]+arr[s]+arr[t];
                if (sum == M) {
                    //만약 sum값이 M보다 동일하면 이후의 어떤 조건을 실행할 필요가 없으므로 종료한다.
                    check = true;
                    result = sum;
                    break;
                }
                //sum이 result(=이전 최대값)보다 크면서 M보다 작은 경우에 result를 새로 갱신해준다.
                if (result < sum && sum<M) {
                    result = sum;
                }

            }
            if(check) break;
        }
        if(check) break;
    }

    System.out.print(result);
