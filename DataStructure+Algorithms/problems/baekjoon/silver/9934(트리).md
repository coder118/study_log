[문제로](https://www.acmicpc.net/problem/9934)

# 문제 정의

> 규칙
1. K로 트리의 깊이가 주어진다.
2. 입력값이 " "을 기준으로 나누어져서 주어진다. 그 값은 inorder 방식(left-root-right)으로 탐색한 순서이다.

> 구해야 하는 정답

- 규칙에 맞는 완전 이진트리를 출력해라. 깊이는 K개이며 \n으로 깊이를 구분한다.

> 문제 해결 흐름도

1. 주어진 숫자값의 규칙성 파악. 중위 우선 탐색을 기준으로 값이 주어진다. L-root-R-root-L-root-R이런 형태이다.
2. L-root-R이 끝나면 항상 root가 나온뒤 다시 LrootR이 반복된다.

1. 2^k-1로 전체 노드 수/2 = root 노드의 인덱스값을 구할 수 있다.(배열에 넣었을 경우)
2. root 인덱스 노드에서 -2를 하면 왼쪽 자식 노드, +2를 하면 오른쪽 자식 노드를 구할 수 있다. 
3. 그 자식노드에서 -1을하면 왼쪽 자식,+1을 하면 오른쪽 자식 노드를 구할 수 있다.

<br>
=> 0,-2,+2,-1,+1이런식의 배열을 구해서 root인덱스에 값을 합치는 식으로 연산을 하는 방법.
<br>
=> 재귀를 통해 root인덱스를 변경하며 접근하는 방법. 근데 이방법은 일렬로 level이 출력되어야 하는 이문제에 적합하지 않은 방법 같다.

<br>
<br>

식을 구해보면 이런 식이 나온다. 
K-(K-K), k-(K+2),K-(K-2),K-(K+1),K-(K-1) 해당 조건을 수행해서 배열을 만든다.

# 다시 해결


    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();
        int K = Integer.parseInt(br.readLine());

        int[] tree = new int[(int)Math.pow(2, K) - 1];
        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int i = 0; i < tree.length; i++) {
            tree[i] = Integer.parseInt(st.nextToken());
        }

        int len = tree.length; //트리의 길이를 저장
        int parent = len; // 기준이 되는 부모 노드를 최고 길이로 저장

        while (parent != 0) {
            int child = (parent-1)/2;
            int gap = (parent-child)*2;
            for (int i = child; i < len; i += gap) {
                sb.append(tree[i] + " ");
            }
            parent = child;
            sb.append("\n");
        }

        System.out.println(sb);

    }

while문을 중심으로 봐야 한다.

1. 트리의 root는 (parent-1)/2라는 식으로 구할 수 있다. 
2. 완전이진트리의 경우는 모든 노드가 다 존재하기 때문에 규칙적인 간격이 있다. 

예> 
- Level 3 (리프): 0, 2, 4, 6 (간격 2)

- Level 2 (중간): 1, 5 (간격 4)

- Level 1 (루트): 3 (간격 없음/8)

3. 이 간격을 만들어서 level1,2,3순으로 출력한다.
4. child 노드에서 gap이라는 간격만큼 이동하면서 tree배열에서 출력한다. level3의 경우는 인덱스 0부터 시작해서 간격 2만큼씩을 가지고 이동한다.
