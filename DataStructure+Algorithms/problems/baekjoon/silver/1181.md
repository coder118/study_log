
[문제로](https://www.acmicpc.net/problem/1181)

# 문제 정의

1. 길이가 짧은 것 부터 출력
2. 길이가 동일할 경우 사전 순 출력
3. 길이가 동일하면서 문자값이 동일 경우 한개 삭제

문자열 배열(문자가 여러개이므로,배열로서 사용될 것)을 주어진 조건을 토대로 **비교**해라. 
<br>
문자열의 길이를 재서 비교할 수 있고 문자열을 하나하나 substring을 하여 뜯어서 비교할 수도 있다.

<br>
<br>

하지만 Arrays.sort에 Comparator의 compare을 사용하면 문자열을 훨씬 편하게 **비교**할 수 있다.

# 코드 풀이


    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    int N = Integer.parseInt(br.readLine());

    String[] arr = new String[N];

    for (int i = 0; i < N; i++) {
        arr[i] = br.readLine();
    }

    Arrays.sort(arr, new Comparator<String>() {
        @Override
        public int compare(String o1, String o2) {
            //문자열 길이가 동일할 경우
            if (o1.length() == o2.length()) {
                return o1.compareTo(o2);
            } else {
                //문자열 길이가 동일하지 않을 경우
                return o1.length()-o2.length();
            }

        }
    });


    StringBuilder sb = new StringBuilder();

    sb.append(arr[0]).append("\n");
    for (int i = 1; i < N; i++) {
        if (!arr[i].equals(arr[i - 1])) {
            sb.append(arr[i]).append("\n");
        }
    }
    System.out.println(sb);

    }

override 받은 함수의 반환값을 보면 int형인 것을 확인할 수 있다. 위 메서드의 반환값은

1. 양의 정수-> 두 객체의 위치를 바꾼다.
2. 0-> 두 객체의 위치를 바꾸지 않는다.
3. 음의 정수 -> 두 객체의 위치를 바꾸지 않는다.

<br>
<br>
위의 특징을 이용해서 문자열의 길이가 동일하지 않을 경우 앞 뒤 문자열의 길이를 비교해서 뒤의 값이 길면 객체의 위치를 바꾸고 아니면 그대로 유지하는 코드를 구현할 수있었다.
