# 문제 정의

처음 문제를 정의했을때 아래와 같았다.

1. 5하고 3으로 나누어떨어지지 않으면 -1을 반환한다.
2. 5로 나누어떨어지면 5로만 합을 구해서 N을 만든다.
3. 3으로 나누어떨어지면 3으로만 합을 구해서 N을 만든다.
4. 3,5로 나누어 떨어지지 않는다면 3과 5를 더해서 N을 만든다. 
    4-1. 최소 횟수를 사용해야 하기 때문에 5를 최대로 더한후 나머지 3을 더해서 N이 나오는지 확인한다.

    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    

    int N = Integer.parseInt(br.readLine());

    int[] bucket = {5,3};
    int sum =0;
    int i =0;
    int count =0;
    while (true) {
        //5하고3으로 나누어지지 않으면 -1 반환
        if (!(N % 5 == 0 || N % 3 == 0)) {
            count = -1;
            break;
        } else if (N % 5 == 0) { //5로만 나누어지면 5만 반복해서 계산
            sum += bucket[0];
            count+=1;
            if (sum == N) {
                break;
            }
        } else if (N % 3 == 0) { //3으로만 반복해서 계산
            sum += bucket[1];
            count+=1;
            if (sum == N) {
                break;
            }
        } else{ //3,5를 섞어서 계산하는데 최솟값이므로 5을 먼저 최대로 더한 후 , 3을 추가로 더한다.
            count+=1;
            //값이 N과 동일하면 종료
            if (sum == N) {
                break;
            } else if (sum > N && i == 1) {
                // 5로 초과되는 경우가 아니라. 3일때 값이 초과되면 종료
                break;
            }
            sum += bucket[i];

            //5로 계속 더한 값이 N보다 크다면 종료하고 3을 더해준다.
            if (sum > N) {
                sum -= bucket[i];
                count-=1;
                i++;
            }
        }

    }

    System.out.println(count);

다시 수정해서 문제를 정의하고 순서도를 다시 구성했다.
문제는 주어진 5,3을 가지고 최소한의 횟수를 사용해서 N을 만드는 것이다. 

1. 5를 먼저 더해준다. N보다 작게 더해준다.
2. 그 값에서 3이랑 더해준다. N값이 나오지 않는다면
3. 5를 하나 빼서 N이 넘지 않게 다시 3을 더해준다.
4. 쭉 반복한다. 5가 다 빠지고 3으로 전부 더해도 나오지 않는다면
5. -1을 반환한다.

# 코드 구현

    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    int N = Integer.parseInt(br.readLine());
    int count5 = N/5; //최대 개수
    int sum;
    int count;
    while (true) {
        sum=N-(count5*5);
        if (sum == 0) {
            //5의 배수로 바로 나누어떨어지는 경우
            count = count5;
            break;
        }
        if (sum % 3 == 0) {
            // 5가 줄어들면서 3으로 나누어떨어지는 경우와 5가 사라지고 3으로 나누어떨어지는 경우
            count = count5 + sum/3;
            break;
        } else if (count5 == 0) {

            //-1을 출력
            count=-1;
            break;
        } else {
            //5를 줄인다.
            count5 -= 1;
        }

    }
    System.out.print(count);

코드를 구현하면서 수정해야 할 부분이 5를 차근차근 더해주고 5를 뺴고 3을 더하다가 3을뺴고 3을 계속 더하는 방식은 코드의 복잡도가 증가해서 다른 방식을 생가했다.

1. N을 5로 나누어 최대 사용할 수 있는 개수를 구한후(count5) 
2. 그 나머지 값이 3으로 나누어떨어진다면 그건 최소한의 횟수를 사용해서 N을 만들어낸 것이다.
3. count5가 0이되고 3으로도 나누어떨어지지 않는다면 N이라는 숫자를 구할 수 없는 것이 된다. 그러므로 -1을 반환한다.

        if (remainingWeight % 3 == 0) {
