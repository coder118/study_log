[문제로](https://www.acmicpc.net/problem/11047)

# 문제 정의

V는 구하고자 하는 값이다.
<br>
주어진 동전을 최소한으로 이용해서 V를 구해라.

1. 최소한으로 이용하려면 큰 동전부터 처리해야 한다.
2. 큰 동전을 더했을때 V를 넘어가면 그 다음으로 작은 숫자를 더한다.
3. 그 합의 값이 V와 동일한 값이 나오면 종료한다.

# 코드 풀이

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int N = Integer.parseInt(st.nextToken());
        int V = Integer.parseInt(st.nextToken());

        int[] coin = new int[N];
        for (int i = 0; i < N; i++) {
            coin[i] = Integer.parseInt(br.readLine());
        }

        //큰 동전 부터 처리하려면 거꾸로 값을 출력해야 한다.
        int sum = 0;
        int count = 0;
        int i = N-1;
        //        coin 배열만큼 반복하느 문제가 아니라. 큰 동전을 사용할 수 잇을때까지 사용해야 한다.
        //총 합이 V랑 같아지면 종료 한다.
        while (sum != V) {
            sum += coin[i];

            if(sum==V){
                count+=1;
                break;
            }
            if (sum < V) {
                count+=1;// 최소 반복을 카운팅한다.
                //가장 큰 수로 더하는 값의 합이 항상 V보다 작아야 한다.
                //이 조건이 실행이 안되는 경우는
                //1000원을 계속 더하다가 V를 초과한 경우이다. 초과했기 때문에 금액을 낮춰야 한다.
                continue;
            }
            sum-=coin[i];// 위에서 더한 값을 빼줘야 한다.
            i--;// 금액을 낮춰야 한다.
        }
        System.out.print(count);
    }

## 중간에 생긴 문제 + 다시 해결한 과정

그냥 간단하게 coin배열을 반복하면 될 것이라고 생각했는데 큰값을 더했을때 V 보다 작으면 그 큰 값을 계속 더해야 최소 횟수로 V를 구할 수 있기 때문에 단순 for문으로는 구현하기 어렵다고 판단했다.

1. 주어지는 값이 오름차순 정렬이 되어 주어진다고 했다. 주어진 값을 배열로 만들고, 가장 큰 값부터 사용할 수 있게 N-1의 인덱스로 접근을 가능하게 했다.
2. 최소 횟수로 V를 만들기 위해 큰 값부터 차례대로 값을 더했다. 그러다가 V를 초과하는 경우에는 i--를 해서 기존 값보다 작은 동전을 사용했다.
3. 값을 더하다가 sum과 V값이 동일해지면 조건을 종료시켰다.

## 다른 사람 코드

    int cnt = 0;
    for(int i = N - 1; i >= 0; i--){
        cnt += V / coin[i];
        V %= coin[i];
        if(V == 0) break;
    }

1. 큰 값부터 반복되게 for문을 만들었다.
2. 나누기를 이용하면 V보다 큰 coin 값은 나누어지지 않게 만들었다.
3. V/coin이 되면 그 몫은 반복한 횟수가 된다. (내가 while으로 하나하나 반복하던 것을 더 간결하게 표현했다.)
4. 나머지로 V값을 갱신하고 V가 0이 되면 종료한다.
