# 문제 정의

> 규칙

1. 왼쪽값은 부모 노드보다 작고 오른쪽은 부모보다 크다.
2. 완전 이진 트리가 아님
3. 전위 우선 순회한 값이 주어진다.

> 구해야 하는 정답

- 후위 우선순회한 값

> 문제해결 로직

1. 전위 우선 순회 (root-L-R)이다. 어디까지가 부모인지를 확인해야한다.
    1-1. 왼쪽값은 항상 작다는 규칙을 이용. 값이 작아지고 있다가 갑자기 커지는 경우가 나오면. 그건 왼쪽까지 탐색이 끝났고 오른쪽 노드로 넘어갔다는 뜻이다.
2. 1번으로 트리를 구한다. 배열의 형태 혹은 map의 형태여도 된다. 
3. 후위 우선 순회 과정의 코드를 작성한다. 

1번 로직 구체화

1. map의 형태로 노드를 재정리. 1번 노드에 value 이런식
2. List<Integer>[] tree을 만들어서 인덱스를 부모 노드로 하고 그 value를 자식 노드로 채운다.
dfs구현 시작. 전위 탐색으로 주어졌고 1~n으로 차례대로 노드가 부여받아진다.
3. min값을 static으로 만들어서 현 노드가 가진 value를 min에 저장한다.
4. min값 보다 현재 노드가 큰 값을 가지고 있으면 왼쪽 자식 노드까지의 탐색이 끝났고 오른쪽 자식 노드로 넘어갔다는 것이 된다.
5. tree에서 오른쪽 자식의 부모 노드를 찾는다. map의 크기 비교를 통해 부모를 찾는다. 부모보다 크고 부모의 부모보다 작아야 한다.

## 문제 해결 로직 재정의

1. Node클래스를 구현한다.
2. 전위 탐색 방식은 (root, root보다 작음, root보다 큼)이렇게 구성
3. Node 클래스 내부에서 왼쪽, 오른쪽에 새로운 Node를 만든다.
4. 들어온 노드 값이 이전 노드보다 작으면 왼쪽, 크면 오른쪽. 
    4-1. 만약 left에 이미 값이 존재하면 재귀로 한번 더 들어간다.
5. 나와서 root의 postorder 재귀함수를 돌려서 출력한다.

> 핵심 로직

    public void insert(int childNum) {
        if (this.num > childNum) { //왼쪽 자식 노드

# 최종 코드

    class Main {

        static class Node{

            int num;
            Node left,right;

            public Node(int num) {
                this.num = num;
            }

            public void insert(int childNum) {
                if (this.num > childNum) { //왼쪽 자식 노드
                    if (this.left == null) {
                        this.left = new Node(childNum);
                    }else{
                        this.left.insert(childNum);
                    }
                }else{//오른쪽 자식 노드
                    if (this.right == null) {
                        this.right = new Node(childNum);
                    }else{
                        this.right.insert(childNum);
                    }

                }
            }
        }

        public static void main(String[] args) throws IOException {
            BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
            String input;
            Node root = new Node(Integer.parseInt(br.readLine()));
            while (true) {
                input = br.readLine();
                if (input == null || input.equals("")) {
                    break;
                }
                root.insert(Integer.parseInt(input));
            }
            postOrder(root);

        }

        public static void postOrder(Node node) {
            if (node == null) {
                return;
            }
            postOrder(node.left);
            postOrder(node.right);
            System.out.println(node.num);
        }

    }
