
# 자료 구조(Data Structure)

데이터의 보관함이다. 

자료구조는 데이터를 어떻게 조직화하고 저장할 것인가에 대해 고민한다.

데이터를 일렬로 세울지(배열,리스트)/ 부모-자식 관계로 쌓을지(트리)/그물망처럼 엮을지(그래프)를 결정하는 것이다.

예> 구글 지도,네비게이션 전 세계 도로를 그래프 구조로 저장하여 두 지점의 최단 거리를 찾는 다익스트라 같은 알고리즘을 사용한다. / 페북,인스타는 사용자간의 관계를 그래프나 트리로 연결하여 좋아할 만한 게시물을 위로 올리는 추천 알고리즘을 사용한다.
